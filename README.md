# My Health Friend



같이 운동할 친구를 모집해주는 어플리케이션입니다.

- 원하는 위치를 지정하여 매칭을 시도할 수 있습니다.
- 프로필을 설정하고 꾸밀 수 있습니다.
- 프로필에 본인의 운동영상을 업로드할 수 있습니다.
- 가까운 순으로 매칭되며, 위치를 변경할 수 있습니다.
